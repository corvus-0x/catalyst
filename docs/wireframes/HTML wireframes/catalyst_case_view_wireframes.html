Three wireframe layout concepts for the Catalyst case investigation view

Concept A
Two-panel — findings + document rail findings center
Documents listed on the left. Findings fill the right. Each finding card shows which documents triggered it inline — you can see the connection without leaving the screen. Clicking a finding highlights its source docs in the left rail.

Thread view — the investigation as a chain discovery order
